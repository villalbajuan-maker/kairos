# Prototype V1 UI

This folder contains an early implementation scaffold for Kairos.

It exists only as a secondary artifact while the main project is being defined through documentation.

## Important

This prototype is not the source of truth for:

- product scope
- theology or biblical handling
- business model
- final architecture

Those foundations live in:

- `/docs`
- `/docs/prd`

## Local run

```bash
npm install
npm run dev
```
